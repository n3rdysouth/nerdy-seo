== Description ==

Nerdy SEO is a comprehensive yet lightweight SEO plugin for WordPress that helps you optimize your website for search engines. It provides all the essential SEO features you need without the bloat.

**Core Features:**

* **Title & Meta Management** - Customize SEO titles and meta descriptions for all your content
* **Social Media Integration** - Open Graph and Twitter Card support for better social sharing
* **Schema Markup** - Automatic JSON-LD structured data for better search results
* **XML Sitemaps** - Automatically generated sitemaps with full control
* **301/302 Redirects** - Manage redirects with 404 tracking
* **Local Business Schema** - Perfect for local businesses with opening hours, location, and contact info
* **Image SEO** - Bulk edit alt text and optimize images
* **Breadcrumbs** - SEO-friendly breadcrumb navigation
* **WooCommerce Integration** - Product schema and SEO optimization for online stores
* **AI-Powered Suggestions** - Generate SEO titles and descriptions with OpenAI or Google Gemini
* **NestedPages Support** - Seamless integration with NestedPages plugin

**Why Choose Nerdy SEO?**

* Lightweight and fast - no performance impact
* Clean, intuitive interface
* No annoying upsells or ads
* Full control over your SEO
* GPL licensed - completely free

**AI Features (Optional)**

Connect your own OpenAI or Google Gemini API key to generate SEO-optimized titles and meta descriptions with AI assistance.

== Installation ==

1. Upload the `nerdy-seo` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Nerdy SEO > Settings to configure your SEO settings
4. Edit individual posts/pages to customize their SEO meta data

== Frequently Asked Questions ==

= Is this plugin free? =

Yes! Nerdy SEO is completely free and GPL licensed.

= Do I need an API key for AI features? =

No, AI features are optional. The plugin works perfectly without any API keys. If you want to use AI-powered content generation, you can connect your own OpenAI or Google Gemini API key.

= Does this work with WooCommerce? =

Yes! Nerdy SEO includes full WooCommerce integration with product schema markup.

= Can I migrate from other SEO plugins? =

Yes, we include a migration tool for All in One SEO (AIOSEO). Go to Nerdy SEO > Migration to import your existing SEO data.

= Does it work with page builders? =

Yes, Nerdy SEO works with all major page builders including Elementor, Beaver Builder, Divi, and more.
